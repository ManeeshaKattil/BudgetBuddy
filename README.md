# 💰 BudgetBuddy – Smart Expense Management System

## 📌 Overview
BudgetBuddy is an intelligent expense management and budgeting application designed to help users track income, manage expenses, and plan financial goals efficiently.

It provides smart insights using forecasting techniques and helps users maintain better financial discipline.

---

## 🎯 Problem Statement
Managing personal finances manually is difficult and error-prone. Users often lack:
- Proper expense tracking
- Budget alerts
- Financial insights

BudgetBuddy solves this by automating expense tracking and providing smart recommendations.

---

## ✨ Features
- 📊 Track income & expenses
- 🔔 Budget alerts & reminders
- 🎯 Goal setting & tracking
- 📈 Expense forecasting
- 💬 Expert chat support
- 📑 Detailed reports
- 🛠️ Admin management system

---

## 👥 Modules
### 🔹 User
- Add income/expense
- Set goals & reminders
- View reports & forecasts

### 🔹 Expert
- Provide financial tips
- Chat with users
- Give suggestions

### 🔹 Admin
- Manage users & experts
- Handle complaints & feedback

---

## 🛠️ Technologies Used
- **Frontend:** Flutter, HTML, CSS, Bootstrap, JavaScript
- **Backend:** Python (Django)
- **Database:** MySQL
- **IDE:** Android Studio, VS Code, PyCharm
